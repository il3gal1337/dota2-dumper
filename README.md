# cs2-dumper

An external offset/interface dumper for Dota 2

## Getting Started

You can download the latest release from [Releases](https://github.com/il3gal1337/dota2-dumper/releases) or compile it yourself.

## Usage

1. Ensure the game is running (Being in the main menu should suffice).
2. Run the `dota2-dumper` executable.
